## Description


    These are the books set in Steven Brust's Dragaera world. These include the books in the [Vlad Taltos](http://www.goodreads.com/series/40334-vlad-taltos) series, as well as those in the [Khaavren Romances](http://www.goodreads.com/series/41289-the-khaavren-romances). The stand-alone novel <i>Brokedown Palace</i> is included here, as it is also set in that world.
