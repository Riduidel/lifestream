# WildC.A.T.s

## Books

 * [WildC.A.T.s: Gang War](../books/9781563895609.md)

## Description


    WildC.A.T.s (Covert Action Teams) collects Wildstorm comic books by the same name.
    
    It includes the following series:
    1. [WildC.A.T.s I](http://www.goodreads.com/series/56178-wildc-a-t-s-i) / [Archivos Wildstorm: WildC.A.T.s](http://www.goodreads.com/series/56184-archivos-wildstorm-wildc-a-t-s)
    2. [WildC.A.T.s II](http://www.goodreads.com/series/56179-wildc-a-t-s-ii)
    3. [WildC.A.T.s 3.0](http://www.goodreads.com/series/56180-wildc-a-t-s-3-0)
    4. (Note: the fourth WildC.A.T.s series is collected entirely in the single book [WildC.A.T.s: Nemesis](http://www.goodreads.com/book/show/1397306.Wildcats).)
    5. [WildC.A.T.s V](http://www.goodreads.com/series/56181-wildc-a-t-s-v)
    
    It also includes standalone books and crossovers, including:
    * [WildC.A.T.s/X-Men](http://www.goodreads.com/series/56185-wildc-a-t-s-x-men)
