## Description


    The original five book series which started the [Belgarian universe](http://www.goodreads.com/series/62114-belgarian-universe). This series is followed by [The Malloreon](http://www.goodreads.com/series/42429-the-malloreon).
    
    For German language editions, series name is "Das Auge Aldurs"
