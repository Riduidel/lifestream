# Goodreads export : yep, another beta reached ! 

Yep, in exactly the same fashion than [the previously mentionned posterous export](http://riduidel.posterous.com/posterous-backup-beta-reached), I've also created a goodreads export script working **exactly** the same way  
The advantages are
1. exactly the same install path (install Java, then groovy, then [download the script](https://dl.dropbox.com/u/2753331/goodreads.groovy))
2. exactly the same command line arguments
3.  exactly the same output XML format (with some extensions for books that I leave to your imagination from the following file)
