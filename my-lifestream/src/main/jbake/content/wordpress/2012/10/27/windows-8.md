# Windows 8 ?

Ben oui, il est sorti.

Clairement, certaines idées sont bonnes, comme par exemple la fameuse interface à tuiles qui remplace le bouton Démarrer (je vous aurais bien mis une image, mais j'ai la flemme).

Je l'ai testée dix secondes dans un quelconque magasin d'électro-ménager, et je dois dire que je suis suffisament séduit pour tester ça de façon plus approfondie. Heureusement, il semble exister au moins un backport de cette interface pour Windows 7 : [Mosaic Project](http://mosaicproject.codeplex.com/).

Et j'ai bien l'impression que c'est une bonne idée, sans doute presque aussi bonne que le [menu radial de World of Tanks](http://worldoftanks.eu/dcont/fb/80_release/commands_radial.jpg) ...
