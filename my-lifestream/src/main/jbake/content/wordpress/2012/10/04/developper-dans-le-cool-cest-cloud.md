# Développer dans le cool, c'est cloud !

Vous savez que depuis un moment, poussé par la fin d'Origo, j'ai dû passer le développement de gaedo chez github ? Eh bien j'ai franchi hier une nouvelle étape : comme c'est un projet full maven, j'ai pu l'inscrire sur [buildhive ](https://buildhive.cloudbees.com/)pour disposer d'[un jenkins dans le cloud](https://buildhive.cloudbees.com/job/Riduidel/job/gaedo/) !

Et je dois dire que je trouve l'idée super chouette !

Pas besoin d'une configuration sophistiquée pour avoir des résultats de tests sous la main. J'ai d'ailleurs été surpris par la rapidité du build Jenkins, qui va quasiment aussi vite que le build sur ma machine de compéition.Du coup, il ne me manque plus qu'un IDE dans mon navigateur pour devenir un vrai développeur cloud. Bon, un IDE, un IDE, je n'ai qu'à passer au [truc d'Exo, là](http://cloud-ide.com/) ... et je pourrais développer de n'importe quelle machine disposant d'un navigateur web et d'un interpréteur Javascript assez rapide.
