# Et un tutorial de fait !

<a>J'ai dit</a> qu'il existait des tutorials faciles d'emploi pour le {Ruby}. Et donc, je m'en suis servi.

Grâce à ce tutorial, je connais donc maintenant les rudiments du langage. Le tutorial est extrêmement bien fait. Il prend l''apprentissage dans un sens excessivement logique : d'abord la manipulation de chaînes, puis les conditions et les boucles, ensuite les fonctions. Une fois ces points de base abordés, on arrive aux tableaux et tableaux associatifs, puis aux classes et méthodes. Malheureusement, ça s'arrête là. Rien sur l'héritage, ni sur les bibliothèques disponibles en Ruby.

Donc, je connais maintenant les bases, et peut comprendre des syntaxes comme `5.times do     puts "mon oeil !" end` Oui, je sais, c'est pas grand chose. Le seul point pénible, dans ce tutorial, c'est qu'un élément clé de la syntaxe Ruby, les "|" dans des accolades, comme par exemple dans ce code : `@personnes.sort { |a, b| par_nom(a, b) }` n'est jamais explicité. On se doute bien qu'il s'agit de transmettre des paramètres à un bloc de code (que j'appelerais anonyme en {Java}). Donc, un tutorial intéressant pour mettre un pied dans la baignoire Ruby.

Pour aller plus loin, c'est clairement insuffisant. Quant au langage, j'en reparlerais ...via [Nicolas Delsaux's posterous import script](http://riduidel.posterous.com/quest-ce-que-cest-que-ce-flux-qui-clignote)
