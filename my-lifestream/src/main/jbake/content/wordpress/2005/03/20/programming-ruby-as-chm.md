# Programming Ruby as CHM

Looking for a good documentation about {Ruby}, I found that there exists a [ProgrammingRuby.chm](http://blog.itpub.net/get/325/ProgrammingRuby.chm) help file for windows. As one may guess, it's a HTML2CHM version of the <span style="text-decoration:underline;">Pragmatic Programmer's Guide</span> that can also be found at [Ruby Central](http://www.rubycentral.com/book/). Great !

En cherchant une bonne documentation sur {Ruby}, j'ai découvert qu'il existait le [ProgrammingRuby.chm](http://blog.itpub.net/get/325/ProgrammingRuby.chm) (en fichier d'aide Windows). Comme on pourrait s'en douter c'est une version HTML2CHM version du <span style="text-decoration:underline;">Pragmatic Programmer's Guide</span> qui peut aussi être trouvé sur [Ruby Central](http://www.rubycentral.com/book/). Génial !via [Nicolas Delsaux's posterous import script](http://riduidel.posterous.com/quest-ce-que-cest-que-ce-flux-qui-clignote)
