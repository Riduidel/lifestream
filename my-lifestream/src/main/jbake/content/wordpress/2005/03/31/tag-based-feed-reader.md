# tag-based feed reader

Il semble finallement que je ne sois pas le seul à croire que la prochaine partie taggée du web sera le RSS. [How Not To Blog](http://hownottoblog.com/index.php/2005/03/30/feed_filtering_via_tags) le dit clairement :

> Wouldn't it be nice to have these thousand feeds, but only display those that are tagged with what I'm looking for? Is there an RSS feed filter?

D'après ce que j'ai cherché précédement (voir les archive de  {Folksonomies}), il semble que tous les lecteurs RSS souscrivent au paradigme feed=entrées. Pour que le concept de flux taggé ait un sens, il faudrait au préalable avoir une espèce de *vrai* aggrégateur, plaçant toutes les entrées du blogroll dans un gros RSS, puis extrayant de ce RSS les entrées correspondant aux tags souhaités (à condition bien s^pur que les tags soient exprimables dans le format RSS). Mmmm... Ne serait-ce pas un nouveau concept de site ?via [Nicolas Delsaux's posterous import script](http://riduidel.posterous.com/quest-ce-que-cest-que-ce-flux-qui-clignote)
