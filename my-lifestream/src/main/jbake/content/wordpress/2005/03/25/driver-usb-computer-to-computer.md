# Driver USB computer-to-computer

  Depuis que je cherche un portable, une question me taraude : n'existe-t-il pas un driver stockage de masse pour PC ? En effet, tout le monde sait que la spécification USB inclut la définition d'un profil de stockage de masse, permettant à un périphérique d'être vu comme une espèce de disque dur. Ne serait-il pas possible d'avoir la même chose pour les ordinateurs, de manière à voir un portable depuis un PC voisin comme un disque dur externe ? Je sais qu'Apple fournit déja un mode de démarrage disque dur Firewire. Dans le même ordre d'idées, j'ai vu une fois un très cher driver réseau pour USB, afin de permettre la connexion directe entre deux PC. N'y a-t-il pas une version free ? C'est une question pour [LazyWeb](http://www.lazyweb.org/).via [Nicolas Delsaux's posterous import script](http://riduidel.posterous.com/quest-ce-que-cest-que-ce-flux-qui-clignote)
