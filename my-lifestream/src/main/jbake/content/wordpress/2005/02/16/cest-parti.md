# C'est parti !

This has already been blogged [there](http://www.43things.com/entries/view/20941). This entry is mainly a french translation of the previous link.  Comme je le dis en anglais [là](http://www.43things.com/entries/view/20941), j'ai commencé à apprendre Ruby avec le [RDT](http://rubyeclipse.sourceforge.net/) (même si il existe [un très bon package pour Windows](http://rubyinstaller.rubyforge.org/wiki/wiki.pl)) et [un tutorial en français](http://www.ruby-doc.org/docs/beginner-fr/xhtml/), très progressif (chapeau aux auteurs). Parmi les choses qui surprennent déja :
* La syntaxe des boucles avec le `5.times do`.
* Les méthodes sans paramètres qui s'écrivent sans aucune parenthèse.
* L'absence complète d'accolades.via [Nicolas Delsaux's posterous import script](http://riduidel.posterous.com/quest-ce-que-cest-que-ce-flux-qui-clignote)
