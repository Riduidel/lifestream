# Some more plugins

  Voici quelques plugins, (différents de [ceux déja présentés](http://nicolas.delsaux.free.fr/wordpress/index.php?p=97)) que je trouve intéressants dans Eclipse :  
* [Un meilleur outil de folding](http://www.coffee-bytes.com/servlet/PlatformSupport)
* [Un puissant (pour moi) système de revue de code](http://csdl.ics.hawaii.edu/Tools/Jupiter/) qui semble s'intégrer très bien avec des contrôles de source comme CVS.
* [Le wiki pour Eclipse](http://eclipsewiki.sourceforge.net/). Je ne sais pas s'il a été porté pour Eclipse 3.0, mais la possibilité de l'exporter vers SnipSnap, et la conversion automatique du code Java en HTML me parle.
* [un système de Collaboration](http://composent.com/plugins/) J'adore ce genre de trucs
* [Un détecteur de bugs](http://findbugs.sourceforge.net/) très utile, surtout quand la qualité des développeurs n'est pas garantie.
* [L'évident détecteur de dépendances cycliques](http://andrei.gmxhome.de/jdepend4eclipse/)
* [Un éviteur de  Ctrl-C Ctrl-V](http://www.blue-edge.bg/download.html)via [Nicolas Delsaux's posterous import script](http://riduidel.posterous.com/quest-ce-que-cest-que-ce-flux-qui-clignote)
