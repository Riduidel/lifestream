# Quelques infos sur les fontes

En guise d'aide mémoire pour l'avenir (lire : le jour où je voudrais utiliser différentes fontes sur ce blog), voici quelques sites intéressants :

* [Web Fonts basics](http://www.efuse.com/Design/web_fonts_basics.html) rappelle quelles fontes sont facilement utilisables sur le web
* [DaFont](http://www.dafont.com/) liste des tonnes plus de polices toutes plus belles les unes que les autres
* Et le [Dynamic Text Replacement](http://www.alistapart.com/articles/dynatext/) devrait me permettre de remplacer le moche code source en Courrier par un joli code source en [Proggy](http://www.proggyfonts.com/index.php?menu=download) avec la mise en couleur syntaxique, pour peu que je me penche (très) sérieusement sur le sujet.
[EDIT]Ce site m'a l'air aussi très bien [UsableType](http://usabletype.com/)via [Nicolas Delsaux's posterous import script](http://riduidel.posterous.com/quest-ce-que-cest-que-ce-flux-qui-clignote)
