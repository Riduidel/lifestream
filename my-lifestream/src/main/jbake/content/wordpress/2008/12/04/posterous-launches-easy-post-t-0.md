# Posterous launches easy post tagging via email

C'est sans oute le truc qui me manquait le plus dans posterous : la possibilité de tagger mes messages histoire de trier un peu.   
D'ailleurs, je vais peut-être même reprendre mes anciennes entrées pour les tagger.   
Le seul inconvénient dans ce cas, c'est que je n'aurais pas les tags dans webgen, à moins bien sûr de supprimer les fichiers dans webgen pour les récupérer d'internet. Oui, mais à ce moment-là, il faudra d'abord que je modifie mon lifestream pour bien gérer les tags de posterous, qui ne seront à tout coup pas gérés de manière standard ...  






##[Posterous launches easy post tagging via email](http://blog.posterous.com/posterous-launches-easy-post-t)via [The Official Posterous Posterous](http://blog.posterous.com) le 22/10/08  


 

We're really excited to be releasing one of our most requested features: tagging! You can now categorize the posts on your site by adding tags either through email, or on the post's page itself. 

  Add tags simply in the subject of your email using the syntax ((tag: apple, gadgets)). Tagging doesn't have to be a tedious task you do after creating a post; simply tag your posts as you create them right in the message. 

  Not only can you see all posts on a site with a given tag, but you can view all posts throughout Posterous with a tag. It's a great way to see what people are saying about [any](http://www.posterous.com/explore/tag/food)[given](http://www.posterous.com/explore/tag/apple)[topic](http://www.posterous.com/explore/tag/obama). Enjoy!
