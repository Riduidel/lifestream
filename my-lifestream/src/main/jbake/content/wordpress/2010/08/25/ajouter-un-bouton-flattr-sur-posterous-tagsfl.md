# Ajouter un bouton flattr sur posterous

Puisque je suis en train de le faire, autant vous l'expliquer. Voici donc ma solution pour ajouter un bouton flattr sur un blog posterous. Pour cela, vous devez déja avoir un compte flattr et un blog posterous.

Ajoutez votre blog comme "thing" flattr. Pour ça, je vous laisse découvrir sur le site de flattr comment faire (en tout fcas, la capture ci-dessous devrait pouvoir vous aider ...).[](http://riduidel.files.wordpress.com/2010/08/ss-2010-08-25_13-38-11.png)Une fois que c'est fait, vous disposerez d'une page comme [celle-ci](https://flattr.com/thing/54138/Riduidels-posterous).

Sur la page de votre blog, il doit y avoir un lien intitulé "<span style="color:#33cc00;"><span style="font-family:tahoma, sans-serif;">Button code</span></span>".[](http://getfile0.posterous.com/getfile/files.posterous.com/riduidel/2eBllS13C414WbyTw0QkYY7IolHRsfCvllZMxzskcrkzvIIS3n3gqbVchvQn/SS-2010-08-25_13.43.57.png)

Cliquez dessus.Dans le panneau qui va s'afficher, vous trouverez deux types de boutons : un bouton dynamique (avec du Javascript, donc, et qui ne marchera pas sur posterous) et un bouton statique (<span style="font-family:tahoma, sans-serif;"><strong>Default static button code</strong></span>). Dans le cadre sous cet intitulé, il y a un paquet de code HTML. Copiez-collez le.[](http://riduidel.files.wordpress.com/2010/08/ss-2010-08-25_13-47-00.png)

Bien, maintenant que vous avez votre code flattr, vous pouvez aller sur votre site posterous. Donc vous vous connectez à posterous, vous cliquez sur settings, et ensuite sur "**Edit Theme**"

[](http://riduidel.files.wordpress.com/2010/08/ss-2010-08-25_13-49-21.png)

Là, vous allez devoir modifier le code HTML du thème de votre posterous. Il faudra donc être prudent ! Pour cela, vous devez d'abord cliquer sur le mode "Advanced" de l'édition de thème posterous.Personnellement, je trouve le bouton bien placé dans le pied de page. J'ai donc modifié le pied de page de mon thème personnel. Hélas, selon le thème utilisé - et vos envies - vous pourrez avoir envie d'autre chose ... Je vous laisse donc cette édition de code HTML en guise d'exercice.
