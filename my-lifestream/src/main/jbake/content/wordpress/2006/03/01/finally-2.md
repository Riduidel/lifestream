# Finally

After years and years of desperate search, there finally seems to exist a free Eclipse plugin able to edit that damned `struts-config.xml` file in a graphical way. Its name is [StrutsIDE](http://amateras.sourceforge.jp/cgi-bin/fswiki_en/wiki.cgi?page=StrutsIDE). It has a strange dependancy over an HTML editing plugin, but nevertheless, here is the magical solution. And, last but not least, it can be used with Eclipse Web Tools !

Après des années et des années de recherche, il semble finallement exister un plugin Eclipse gratuit capable d'éditer ce maudit fichier `struts-config.xml` graphiquement. Son nom est [StrutsIDE](http://amateras.sourceforge.jp/cgi-bin/fswiki_en/wiki.cgi?page=StrutsIDE). Il a une dépendance bizarre sur un plugin d'édition HTML, mais c'est néanmoins la solution magique. Et, cerise sur le gateau, il peut être utilisé avec Eclipse Web Tools !via [Nicolas Delsaux's posterous import script](http://riduidel.posterous.com/quest-ce-que-cest-que-ce-flux-qui-clignote)
